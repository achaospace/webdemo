    • 开始使用 - 入门指南

    byui基于gulp构建的前端框架，是js模块化开发解决方案。目前我们PC端、手机端项目前端都是基于jquery项目构建，项目中前端引用框架、插件过多，并且并不是全部框架功能都有使用，严重影响前端页面加载速度，鉴于以上原因，把常用的功能基于layui重新重构了一下，打算用一个框架。保留了layui样式，删除了暂时用不到的代码，取其精华。希望对以后项目有所帮助。
    兼容性和面向场景
    byui 不兼容IE6/7。
    快速上手 
    模块化引用
    1. ./byui/css/byui.css
    2. ./byui/byui.js 
    2. <script src="../byui/byui.js"></script>
    3. <script>
    4. //一般直接写在一个js文件中
    5. byui.use(['jquery','byscroll'], function(){
    6.   var $= byui.$
    7.   ,byscroll = byui.byscroll;
    8. console.log($) 
    10. });
    11. </script>

    我们推荐你遵循 byui 的模块规范建立一个入口文件，并通过 byui.use() 方式来加载该入口文件，如下所示：

    1. <script>
    2. byui.config({
    3.   base: '/res/js/modules/' //你存放新模块的目录，注意，不是byui的模块目录
    4. }).use('index'); //加载入口
    5. </script>    
    6.
    上述的 index 即为你 /res/js/modules/ 目录下的 index.js，它的内容应该如下： 
    1. /**
    2.   项目JS主入口
    3.   以依赖byui的jquery和byscroll模块为例
    4. **/    
    5. byui.define(['jquery', 'byscroll'], function(exports){
    6.   var jquery= byui.$
    7.   ,byscroll = byui.byscroll;
    8.   
    9.   console.log($)
    10.   
    11.   exports('index', {}); //注意，这里是模块输出的核心，模块名必须和use时的模块名一致
    12. });
